## [백준] A-B ##

[원본 문제](https://www.acmicpc.net/problem/1000)   


### 문재 풀이1 ###
```kotlin
import java.io.BufferedReader
import java.io.InputStreamReader

fun main() = with(BufferedReader(InputStreamReader(System.`in`))) {

    val (a: Int, b: Int) = readLine()
        .split(" ")
        .map { it.toInt() }

    println(a - b)

}
```


### 문제 풀이2 ###
```kotlin
import java.io.BufferedReader
import java.io.InputStreamReader

fun main() = with(BufferedReader(InputStreamReader(System.`in`))) {

    println(
        readLine()
            .split(" ")
            .map { it.toInt() }
            .reduce{ a, b -> a - b }
    )

}
```
