## [백준] A-B ##

[원본 문제](https://www.acmicpc.net/problem/1000)   

```kotlin
import java.io.BufferedReader
import java.io.InputStreamReader

fun main() = with(BufferedReader(InputStreamReader(System.`in`))) {

    val ab = readLine()
        .split(" ")
        .map { it.toInt() }

    println(ab[0] - ab[1])

}
```
