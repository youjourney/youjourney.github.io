### 문제 설명
[백준 15650번 문제 링크](https://www.acmicpc.net/problem/15650#description){:target="_blank"}




### 입력 및 출력
#### >> 입력
첫째 줄에 자연수 N과 M이 주어진다. (1 ≤ M ≤ N ≤ 8)



#### >> 출력
* 한 줄에 하나씩 문제의 조건을 만족하는 수열을 출력한다. 중복되는 수열을 여러 번 출력하면 안되며, 각 수열은 공백으로 구분해서 출력해야 한다.
수열은 사전 순으로 증가하는 순서로 출력해야 한다.





### 예제 입출력(테스트케이스)


|입력|출력|
|-----|------|
|3 1|1<br>2<br>3|
|4 2|1 2<br>1 3<br>1 4<br>2 3<br>2 4<br>3 4|
|4 4|1 2 3 4|




### 문제 풀이(SWIFT) 1
```swift
//
//  main.swift
//  BOJ15650_SWIFT
//
//  Created by choiyoujun on 2022/02/14.
//

let NM = readLine()!.split(separator: " ").map { Int($0)! }
let N = NM[0]
let M = NM[1]
var check = [Bool](repeating: false, count: N + 1)
var stack = [Int]()

func result(S: Int, depth: Int) -> Void {
    if depth == M {
        print(stack.map { String($0) }.joined(separator: " "))
        return
    } else {
        if S <= N {
            for i in S...N {
                if !check[i] {
                    check[i] = true
                    stack.append(i)
                    result(S: i + 1, depth: depth + 1)
                    check[i] = false
                    stack.removeLast()
                }
            }
        }
    }
}

result(S: 1, depth: 0)
```
