### 문제 설명
[백준 2420번 문제 링크](https://www.acmicpc.net/problem/2420#description){:target="_blank"}




### 입력 및 출력
#### >> 입력
첫째 줄에 두 도메인의 유명도 N과 M이 주어진다. (\-2,000,000,000 ≤ N, M ≤ 2,000,000,000)



#### >> 출력
첫째 줄에 두 유명도의 차이 (|N\-M|)을 출력한다.





### 예제 입출력(테스트케이스)


|입력|출력|
|-----|------|
|\-2 5|7|




### 문제 풀이(SWIFT) 1
```swift
//
//  main.swift
//  BOJ2420_SWIFT
//
//  Created by choiyoujun on 2022/01/20.
//

let value = readLine()!
    .split(separator: " ")
    .map { Int($0)! }

print(
    abs(
        value[0] - value[1]
    )
)

```
