### 문제 설명

정수 num이 짝수일 경우 Even을 반환하고 홀수인 경우 Odd를 반환하는 함수, solution을 완성해주세요.

### 제한 사항

  * num은 int 범위의 정수입니다.
  * 0은 짝수입니다.

### 예제 입출력

|num|return|
|-|-|
|3|"Odd"|
|4|"Even"|

### 문제 풀이1

```kotlin
class Solution {
    fun solution(num: Int): String = if (num % 2 != 0) "Odd" else "Even"
}
```
